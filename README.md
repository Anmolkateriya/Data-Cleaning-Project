# Data Cleaning & Preparation

## Objective
Clean a raw dataset by handling missing values, duplicates, and incorrect data formats using Python.

## Tools Used
- Python
- Pandas
- OpenPyXL

## Tasks Performed
- Identified missing values
- Filled missing CouponCode values with "No Coupon"
- Checked and removed duplicate rows
- Verified data formats
- Saved the cleaned dataset

## Files
- Dataset for Data Analytics.xlsx
- Cleaned_Dataset_for_Data_Analytics.xlsx
- data_cleaning.py

## Output
A cleaned Excel dataset ready for analysis.
